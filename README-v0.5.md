# PhotographyCourses.com Virtual Camera Lab v0.5

## Files
- `index.html`
- `style.css`
- `camera.js`

## New in v0.5
- Three learning scenes: portrait, moving cyclist and night street
- Camera modes: Manual, Aperture Priority, Shutter Priority and Auto
- Lens selection from 24mm to 200mm
- Exposure, depth-of-field, motion blur and ISO-noise simulation
- Scoring and scene-specific instructor feedback
- Recent photo history
- Optional shutter sound
- Keyboard controls
- Responsive mobile design

## Upload to GitHub
Replace the existing root files in the repository with these three files, then commit the changes.
GitHub Pages should update automatically.
